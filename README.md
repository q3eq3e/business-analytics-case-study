# Sales, product evolution and logistics in XXX: Integrating industrial and data science perspectives

## Overview
This repository presents a portfolio-ready version of a real-world data analysis project.  
It focuses on the process of acquiring, cleaning and transformating a structured dataset of potential clients and analyzing it to generate actionable insights.  
All data used here has been anonymized, and sensitive information has been removed.

## Repository Structure
- `Creating market dataset.Rmd` – Describes the process of creating the final dataset from publicly available and structured sources.  
- `Analysing market data.Rmd` – Contains exploratory data analysis, statistical summaries, and derived insights based on the prepared dataset.  
- `Memoria.pdf` – A summary report highlighting project objectives, methodology, analysis, and data transformations.

## How to Use
1. Open the `.Rmd` files in RStudio to review code and methodology.  
2. Knit the `.Rmd` files to generate HTML or PDF outputs.  
3. Refer to `Memoria.pdf` for a high-level summary of the analysis and conclusions.

## Notes
- Dataset contains only anonymized, non-sensitive information.  
- The project is intended as a portfolio example demonstrating data analysis and business insight skills.
